# Pokemon Shop

This ReactJS/Redux application is a shopping website where you can add pokemons to your cart and make an order (for
 free of
 course).

## Install

- Install [Node.js](https://nodejs.org/en/) (use [nvm](https://github.com/nvm-sh/nvm))

```bash
npm i
```

## Run

```bash
npm start
```
